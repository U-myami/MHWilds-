●目的

　=> コンシューマー、ソーシャルゲーム、アーケードなど、様々なジャンルのゲーム攻略サイトを作成する
　=> ポートフォリオとして作成
　=> 前回の失敗を活かす


●ターゲット設計

　=> ■初心者ペルソナ

　　  ・ 過去作経験無し
　　  ・ 武器選びで迷っている
　　  ・ ストーリーの進め方

　   ■復帰勢ペルソナ

　　  ・ 過去作経験有り
　　  ・ システム変更点を知りたい
　　  ・ 今の環境装備を知りたい

　   ■上級者ペルソナ

　　  ・ モンスターの耐久値、気絶値計算
　　  ・ 部位ごとのダメージ計算


●市場調査・競合分析

　=> ・ Youtuber名
　　 ・ 武器種別おすすめ装備TOP3
　　 ・ 生存特化 or 火力特化比較
　　 ・ 立ち回りと装備紹介動画埋め込み
　　 ・ DPS比較(理論値)
　　 ・ 初心者向けのテンプレ装備


●コンセプト

　=> 誰でもゲームの世界観と同じUIで使いやすい攻略サイト

　　 見やすさ特化  : 最初から必要な条件で絞ることで余計な情報を目に入れなくて良くなる
　　 初心者特化    : 何からすればいいか、おすすめ装備を強調することでサイトで迷子にならずに良くなる
　　 画像<=動画    : 画像より動画がいいのは実際のゲームでの体感を味わえるほか、全体が見えるようになる
　　 世界観特化    : トップページはゲームのキャラや背景などを多すぎないように配置
　　　　　　　　　   ボタンはランダムで木製UI、鉄製UIなどバリエーションを持たせること
　　　　　　　　　   色合いやフォントは世界観でよく使われている色合い、またはキャンプエリアの雰囲気の色合い
